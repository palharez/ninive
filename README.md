# Sgb :orange_book:

## Installation

You can install `sgb` via [pipenv](https://github.com/pypa/pipenv) from [pypa](https://github.com/pypa):

```shell
$ pipenv install --dev
```

## Usage

For linux and Mac:

```shell
$ export FLASK_APP=sgb
$ flask run
```

## Modules

-   [Flask](http://flask.pocoo.org/docs/1.0/#)
-   [MySQL](https://www.mysql.com/)
